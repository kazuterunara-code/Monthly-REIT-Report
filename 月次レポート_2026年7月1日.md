**最新会合：** 2026年6月15日（月）・16日（火）

**決定内容：** 政策金利を **+0.25%引き上げ（0.75% → 1.00%）**

**次回会合予定：** 2026年7月30日（木）・31日（金）

## 3. 東証REIT指数 直近1ヶ月の動向

### ①「Jリート市場の現状と見通し：2026年6月」（しんきんアセットマネジメント投信、2026年6月）

5月の東証REIT指数は前月末比 **-3.8%（1,809.65ポイント）**。米・イラン情勢緊迫化による原油高と国内長期金利の急上昇、日銀6月利上げ観測の高まりが重なり、月末にかけて下落が続いた。  
6月予想レンジ：**1,700〜2,000ポイント**（1年間見通し）。分配金利回りが5%程度まで上昇しており、下値は限定的と見込む。

### ②「J-REIT市場 現状と今後の見通し（2026年6月号）」（ニッセイアセットマネジメント、2026年6月9日）

5月末の主要指標：

| 指標 | 数値 |
|------|------|
| 東証REIT指数（5月末） | 1,809.65ポイント（前月末比-3.76%） |
| 予想配当利回り | **5.07%** |
| 10年国債利回り | 2.66% |
| イールドスプレッド | **2.41%** |
| NAV倍率 | **0.85倍**（過去平均1.04倍を42ヵ月連続で下回る） |
| J-REIT保有物件含み率 | **27.8%**（算出開始以来最高水準） |

2026年の年初来パフォーマンスはTOPIX（+16.08%）に対しREIT（-10.12%）と大幅に出遅れ。AI需要を背景とした国内株への資金シフトも重しに。  
一方、物件の含み益は過去最高水準を維持しており、ファンダメンタルズは堅調。オフィス賃料（都心5区）は前年同月比+8.19%と24ヵ月連続上昇。

### ③「日銀、予想通り利上げを決定 次の利上げは12月メイン」（野村證券・森田京平、2026年6月）

6月会合での利上げを「予想通り」と評価。次回利上げのメインシナリオは **2026年12月**、10月はリスクシナリオとして位置づける。  
J-REITは金利敏感セクターとして引き続き上値が重い展開が続く見通し。一方で、賃料上昇が継続する良好な不動産市況を背景に、利回り水準の魅力を評価した買いも期待される。
